Essential Tab is a minimal, widget-based Chrome extension inspired by the Nothing OS design language. It replaces your default new tab page with a clean, focused, and highly customizable dashboard.

## Features

### Morning Brief
A dedicated summary overlay featuring weather, tasks, and daily quotes.

### Smart Widgets
Includes multiple productivity-focused widgets:
- Countdown timer (customizable, suitable for exams like JEE)
- Recent tabs
- Clock

### Media Control
An integrated media player interface that allows you to see and control what is currently playing in your browser.

### Quick Dock
A centered app dock for fast navigation to commonly used sites such as:
- Spotify
- YouTube
- ChatGPT

## Installation

![Installation Steps](https://github.com/user-attachments/assets/1cdd9742-c326-4e41-88a9-aca2fc437bd7)

1. Download the ZIP  
   Click the green "Code" button, select "Download ZIP", and extract the folder.

2. Open the Extensions page  
   Navigate to chrome://extensions/ or edge://extensions/.

3. Enable Developer Mode  
   Toggle the Developer Mode switch in the top-right corner.

4. Load the project  
   Click the "Load unpacked" button and select the folder containing the manifest.json file.

5. Done  
   Open a new tab to see your new dashboard.
